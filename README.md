# Rental Portfolio Financial Dashboard

A multi-sheet Excel financial model tracking 11 residential + 2 dummy properties across 7 metrics — including NOI, cap rate, and budget vs. actual variance — with an automated summary dashboard and live transaction logging.

## Tools
Microsoft Excel (PivotTables, XLOOKUP, SUMIFS, cross-sheet referencing)

## Data
Real residential property portfolio data, addresses and tenants anonymized for privacy

## Key Features
- Automated summary dashboard that updates across sheets in real time
- Transaction log tracking rent collection and expenses by property and month
- NOI, cap rate, occupancy rate, and budget vs. actual variance calculations
